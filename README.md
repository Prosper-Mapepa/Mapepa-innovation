# Mapepa Innovation — Marketing Website

A modern, professional marketing website for **Mapepa Innovation LLC**, a technology and software company offering B2B and B2C products and services.

## Tech Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (animations)
- **Lucide React** (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Adding VeriTalent Screenshots & Videos

Place your app assets in `public/veritalent/`:

```
public/
  veritalent/
    home.png
    profile.png
    search.png
    verify.png
    demo.mp4
```

Then update the `VeriTalent` component to reference them.

## Build for Production

```bash
npm run build
npm start
```

## Deploy

Deploy to [Vercel](https://vercel.com), Netlify, or any platform that supports Next.js.
